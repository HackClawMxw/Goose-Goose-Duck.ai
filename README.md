# Goose-Goose-Duck.ai 🦢🦆

## 项目简介

Goose-Goose-Duck.ai 是一个基于 **LLM Agent + 多智能体仿真 + 游戏逻辑** 的混合系统，旨在通过 AI 智能体模拟经典的社交推理游戏（类似狼人杀/鹅鸭杀）。

本项目不仅是一个游戏Demo，更是一个完整的**多智能体仿真平台**，涵盖了现代 AI 应用开发的核心技术栈。

---

## 核心特性

### 🤖 LLM Agent / 多智能体架构 ✅
- **任务规划与角色扮演**：根据场景目标生成行动策略（伪装、投票等）
- **记忆与知识管理**：记住对话历史、推断身份、维护信念状态
- **多智能体协调**：实现 Agent-to-Agent 通信协议与协作/竞争模式
- **LLM支持**：支持 GLM、OpenAI、DeepSeek 等多种大模型

### 🗺️ 多智能体仿真 + 空间逻辑（规划中）
- **空间索引**：使用网格索引、四叉树等空间数据结构，快速查询视野内的 Agent
- **感知与可见性**：视野范围计算、遮挡关系处理
- **仿真调度**：离散事件仿真，按回合执行移动、对话、行动、投票等阶段

### 🧠 记忆、知识图谱与 RAG
- **短期记忆** ✅：当前对话历史与已知信息
- **长期记忆**（规划中）：历史轮次的证词与行为记录
- **向量数据库**（规划中）：使用 RAG 技术检索相关记忆
- **知识图谱**（规划中）：结构化存储事实与推断关系

### ⚙️ 工程与系统设计 ✅
- **状态管理**：全局世界状态、玩家存活状态、投票结果等
- **游戏流程**：自动化的游戏流程（讨论→投票→处决）
- **配置管理**：支持 YAML 配置文件和环境变量
- **日志系统**：详细的日志记录和调试信息

### 📊 评测与运营（规划中）
- **游戏平衡**：角色胜率统计、参数调优
- **自动化评测**：大规模模拟对局、策略分布分析
- **可视化系统**：地图展示、Agent 行为轨迹

## 技术栈

### 核心框架
- **Agent 框架**：LangChain / LangGraph, AutoGen, LlamaIndex
- **LLM 服务**：支持多种大语言模型（GLM-5 等）
- **向量数据库**：用于 RAG 和记忆检索
- **图数据库**：用于知识图谱构建

### 仿真引擎
- **空间索引**：网格索引、四叉树、KD-tree
- **事件调度**：离散事件仿真框架
- **状态机**：行为树、有限状态机

### 工程工具
- **后端服务**：Python (FastAPI/Flask)
- **可视化**：Web 前端展示
- **监控**：日志系统与性能监控

## 快速开始

📖 **详细教程请查看 [快速开始指南](QUICKSTART.md)**

### 快速运行

```bash
# 1. 克隆项目
git clone https://github.com/yourusername/Goose-Goose-Duck.ai.git
cd Goose-Goose-Duck.ai

# 2. 安装依赖
pip install -r requirements.txt

# 3. 配置API密钥
cp .env.example .env
# 编辑 .env 文件，填入你的API密钥（GLM_API_KEY 或 OPENAI_API_KEY）

# 4. 运行游戏
python main.py
```

### 示例代码

查看 `examples/` 目录了解更多用法：
- `basic_game.py` - 基础游戏示例
- `custom_game.py` - 自定义游戏示例
